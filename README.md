cv
==

## 0. Quick start

```
lualatex --shell-escape cv.tex
```

or

```
xelatex --enable-write18 cv.tex
```

## 1. Prerequisites

On Debian, install the following packages:
- texlive-science
- texlive-extra-utils
- texlive-fonts-extra
- gawk (required by ./vc.sh)

## 2. Compile

```
lualatex --shell-escape cv.tex
```

or

```
xelatex --enable-write18 cv.tex
```
